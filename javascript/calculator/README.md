Calculator
---
This was my first proper project using Javascript as part of the Odin Project!

📦 Tech Stack
---
* HTML
* CSS
* Javascript

🕶️ Features
---
Here's what you can do with Calculator:
* **Add**: Add any two numbers.
* **Subtract**: Subtract any two numbers.
* **Multiply**: Multiply any two numbers.
* **Divide**: Divide any two numbers and it will show up to 5 dp.
* **Clear**: Clear your result.

 👩‍🍳 The Process
---
I started by building out the buttons with HTML and CSS. I also made the numbers show up on the display!

Then I started building out the functions for adding,subtracting,multiplying and dividing.

Storing the results and clearing functions came next. This took quite a bit of debugging as it took me some time to work with the eventlisteners to get my functions right.
